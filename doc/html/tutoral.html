GPTwoSample step by step Tutorial¶

gptwosample.py is a tool to run two-sample tests on time series differential gene expression experiments. The calling signature is:

gptwosample [-h] [-o DIR] [-t] [-c N] [-p] [-v]
[–version] [–backend [PDF,...]] FILE FILE

where:

positional arguments:
FILE treatment/control files to compare against each other
optional arguments:
-h, --help show this help message and exit
-o DIR, --out DIR
 set output dir [default: ./twosample_out/]
-t, --timeshift
 account for timeshifts in data [default: False]
-c N, --confounder N
 account for N confounders in data [default: 0]
-p, --plot plot data into outdir/plots? [default: False]
-v, --verbose set verbosity level [default: 0]
--version show program’s version number and exit

–backend [PDF,...] matplotlib backend - see matplotlib.use(backend)

Data format of FILE:¶

The format of the two .csv files (FILE FILE in usage) is as follows:

arbitrary x1 ... xl
Gene Name 1 y1 replicate 1 ... yl replicate 1
... ... ... ...
Gene Name 1 y1 replicate k1 ... yl replicate k1
...      
Gene Name n y1 replicate 1 ... yl replicate 1
... ... ... ...
Gene Name n y1 replicate kn ... yl replicate kn
